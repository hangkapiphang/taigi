## ðŸš€ How to Install (GitHub Pages)

To play this game, it must be hosted on a secure server (HTTPS) for the microphone to work. GitHub Pages is free and perfect for this.

1.  **Create a Repository:**
    *   Go to GitHub and create a new public repository (e.g., `voice-rush`).

2.  **Upload Files:**
    *   Upload `index.html`, `style.css`, and `script.js`.
    *   (Optional) Upload `class_data.json` if you want a default list.

3.  **Enable Pages:**
    *   Go to **Settings** > **Pages**.
    *   Set **Source** to `main` branch.
    *   Click **Save**.

4.  **Play:**
    *   Wait 1 minute.
    *   Your game will be live at: `https://<your-username>.github.io/voice-rush/`
